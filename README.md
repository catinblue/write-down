# write-down

A single-page browser writing app.

## Quick Start

    npm install
    npm run dev

Open the URL Vite prints.

## Build

    npm run build

Output goes to `dist/` — deploy to any static host (Netlify, Cloudflare Pages, Vercel, plain nginx). No backend required.

## Tech

React 19 · Vite 7 · TypeScript 5.7 · Tailwind CSS 4 · WebGL2.

## Requirements

Node.js ≥ 20 and a modern browser with WebGL2 support.
